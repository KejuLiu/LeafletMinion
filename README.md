# LeafletMinion
LeafletMinion是基于Javascript的Leaflet库开发，UI界面在demos文件夹中，实现轻量级的交互地图操作，提供的图层为高德地图、ESRI卫星影像、天地图矢量图、天地图地形图、天地图卫星影像。
